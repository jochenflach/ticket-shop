Handbuch & Dokumentation

Ticketshop "Das Wilde Weib" — Systemverwaltung

Status: Live / Online

Version: 1.0 (PostgreSQL)

Dieses Handbuch beschreibt die Einbindung auf Ihrer Homepage, den Betrieb am Tag der Veranstaltung (inkl. des ausfallsicheren Offline-Modus), die Verwaltung von Passwörtern (PINs) sowie die Bedienung des Admin-Dashboards.

1. Einpflegen des Shops auf eine Webseite

Da der Ticketshop auf dem hochverfügbaren Servernetzwerk von Netlify gehostet wird, binden wir ihn per Iframe (Fenster-in-Fenster) auf Ihrer Strato-, Hostinger- oder WordPress-Webseite ein. Dadurch läuft der Shop direkt in Ihrem Design.

HTML-Einbettungscode:

Erstellen Sie eine neue Unterseite auf Ihrer Homepage (z. B. tickets.html) und fügen Sie im HTML-Editor folgenden Code ein:

<iframe 
  src="https://wondrous-gumdrop-b59508.netlify.app/?embed=true" 
  style="width: 100%; height: 950px; border: none; overflow: hidden;"
  scrolling="no"
></iframe>
💡

Wie funktioniert ?embed=true? Dieser Parameter signalisiert dem Ticketshop, dass er eingebettet wird. Er schaltet automatisch den großen Hintergrundbanner aus und optimiert das Layout, damit der Shop wie ein nativer Bestandteil Ihrer eigenen Webseite aussieht.

2. Einrichten der Offline-Funktionalität (Veranstaltungstag)

Bei Veranstaltungen fallen Mobilfunknetze und Internetverbindungen oft aus. Um einen absolut reibungslosen Einlass zu garantieren, besitzt das System einen lokalen Offline-Modus. Alle Handys der Ticket-Scanner gleichen die Codes dann direkt mit Ihrem Laptop ab.

Benötigte Hardware vor Ort:

  • Laptop (auf dem dieses Programm installiert ist).
  • Standard WLAN-Router (dieser muss nicht mit dem Internet verbunden sein).
  • Smartphones der Helfer für den Einlass-Scan.

Schritt-für-Schritt-Ablauf am Einlasstag:

1 Daten aus der Cloud laden (Zuhause mit Internet)

Bevor Sie zum Veranstaltungsort aufbrechen, müssen Sie die aktuellen Buchungen aus der Cloud auf Ihren Laptop laden:

  1. Öffnen Sie den Projektordner auf Ihrem Laptop.
  2. Klicken Sie doppelt auf die Datei Ticketshop-Starten.bat.
  3. Drücken Sie die Taste 1 auf Ihrer Tastatur und bestätigen Sie mit Enter.
  4. Das Skript lädt nun alle Tickets und Reservierungen herunter. Sobald es fertig ist, können Sie das Fenster schließen.

2 Lokales WLAN aufbauen (Am Veranstaltungsort)

Bauen Sie vor Ort das Offline-Netzwerk auf:

  1. Schalten Sie den WLAN-Router ein.
  2. Verbinden Sie Ihren Laptop mit dem WLAN dieses Routers.
  3. Verbinden Sie die Smartphones aller Einlass-Helfer ebenfalls mit diesem WLAN.

3 Offline-Server starten

Starten Sie das lokale System auf Ihrem Laptop:

  1. Klicken Sie doppelt auf die Datei Ticketshop-Starten.bat.
  2. Drücken Sie die Taste 2 und bestätigen Sie mit Enter.
  3. Ihr Laptop stellt sich auf die Offline-Datenbank um, startet den Server und öffnet das Verkäufer-Terminal im Browser.

4 Scanner-Handys verbinden

Richten Sie die Handys Ihrer Helfer ein:

  1. Auf dem schwarzen Konsolenfenster Ihres Laptops wird Ihre aktuelle IP-Adresse angezeigt (z. B. http://192.168.1.50:3000/scan).
  2. Die Helfer tippen diesen Link in den Browser (Safari/Chrome) ihres Handys ein.
  3. Nach Bestätigung des Kamerazugriffs können die Helfer Tickets scannen. Alle Scans werden in Echtzeit auf Ihrem Laptop abgeglichen!

3. Ändern der Passwörter (PINs)

Der Ticketshop besitzt zwei Sicherheits-PINs: Die Admin-PIN (Standard: 4321) für Gutscheine und Buchungslisten sowie die Seller-PIN (Standard: 1234) für das Kassenpersonal.

A. PINs für die Webseite (Netlify) ändern:

  1. Loggen Sie sich auf Netlify.com ein.
  2. Navigieren Sie zu Ihrem Projekt und klicken Sie im Menü links auf Site configuration -> Environment variables.
  3. Suchen Sie nach dem Eintrag ADMIN_PIN bzw. SELLER_PIN.
  4. Klicken Sie auf das Menü (drei Punkte) daneben, wählen Sie Edit, tragen Sie den neuen Wert ein und speichern Sie.
  5. Gehen Sie links im Menü auf Deploys, klicken Sie auf Trigger deploy -> Deploy project without cache. Nach ca. einer Minute sind die neuen PINs auf Ihrer Live-Webseite aktiv.

B. PINs für den Offline-Laptop ändern:

  1. Öffnen Sie im Projektordner des Laptops die Konfigurationsdatei .env mit einem Texteditor.
  2. Ändern Sie die PINs in den entsprechenden Zeilen: 
    ADMIN_PIN="IhreNeueAdminPin"
SELLER_PIN="IhreNeueSellerPin"
  3. Speichern und schließen Sie die Datei. Die neuen PINs sind ab dem nächsten Start über die .bat-Datei lokal aktiv.

4. Bedienung des Admin-Dashboards

Das Admin-Dashboard erreichen Sie unter https://ihre-seite.netlify.app/admin (bzw. offline unter http://localhost:3000/admin).

Gutscheincodes anlegen und verwalten:

Sie können flexibel Gutscheincodes erstellen, um Rabatte zu gewähren. Das System unterstützt zwei Gutschein-Typen:

Gutschein-Typ Funktion Beispiel
PERCENT (Prozentual) Zieht einen prozentualen Rabatt von der Gesamtsumme ab. Wert: 20 zieht 20% ab (z.B. 8,00 € Ersparnis bei 40 € Ticketpreis).
FIXED (Festbetrag) Zieht einen festen Euro-Betrag von der Gesamtsumme ab. Wert: 5.00 zieht exakt 5,00 € vom Gesamtpreis ab.

Schritte zum Erstellen eines Gutscheins:

  1. Melden Sie sich mit Ihrer Admin-PIN im Dashboard an.
  2. Klicken Sie auf die Schaltfläche "Gutscheincode hinzufügen".
  3. Geben Sie den Wunschnamen ein (z. B. RABATT2026).
  4. Wählen Sie den Typ (PERCENT oder FIXED) und geben Sie den Wert ein.
  5. Geben Sie optional bei Max. Einlösungen ein Limit ein (z. B. 50, wenn der Gutschein nur für die ersten 50 Käufe gelten soll). Lassen Sie das Feld leer für unbegrenzte Nutzungen.
  6. Klicken Sie auf "Gutschein erstellen". Der Gutschein ist ab sofort live im Online-Shop und an der Abendkasse nutzbar!
ℹ️

Gutscheine löschen: Wenn eine Aktion abgelaufen ist, können Sie den Gutscheincode in der Admin-Tabelle ganz einfach mit einem Klick auf "Löschen" dauerhaft deaktivieren.
